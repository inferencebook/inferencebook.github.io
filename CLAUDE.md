# Project: Inference in Statistical Modelling and Machine Learning

Companion website for the CUP textbook by Burridge and Tosh (inferencebook.github.io). Static HTML served via GitHub Pages — no build step.

## Memory

Persistent memory for this project is stored in `.claude/memory/` (git-tracked, shared across machines). At the start of each session, read `.claude/memory/MEMORY.md` for the index, then load relevant memory files. When saving new memories, write to `.claude/memory/` and update `.claude/memory/MEMORY.md`. Include `machine: <hostname>` in frontmatter for machine-specific memories (e.g. file paths), or `machine: all` for general memories.
