This repo (inferencebook.github.io) is the companion page for the Cambridge University Press book "Inference in Statistical Modelling and Machine Learning".

Authors: Burridge and Tosh.

**Current work (as of 2026-05-22):** James is reviewing the notebooks chapter by chapter, checking consistency with the book text and that exercises make sense.

**Why:** Provides supplementary materials (notebooks, data, PDFs) for readers of the textbook.

**How to apply:** Changes to the site should support the book's pedagogical purpose. The site is static HTML — no build step, served via GitHub Pages.
